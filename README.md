Minesweeper (Python CLI Game)

使用 Python 套件製作的命令列打地雷遊戲，透過終端機即可輕鬆遊玩！

🎮 功能特色

使用 Python   套件製作

透過輸入 x y游標，輕鬆遊玩

傳統經典的打地雷遊戲玩法

適合終端機環境，快速輕巧

🚀 開始遊戲

確認電腦已安裝 Python（3.7 以上版本推薦）

啟動遊戲：

python minesweeper.py

🎯 操作方式

使用輸入 x y游標， 揭開地雷區域。

 

💻 環境需求

Python 3.7 或更新版本
